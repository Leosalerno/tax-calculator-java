# Tax Calculator - Java

Projeto simples em Java que calcula impostos de contribuintes com base em:

- Renda de salário
- Renda de serviços
- Ganho de capital
- Gastos médicos
- Gastos educacionais

## Conceitos usados

- Programação Orientada a Objetos
- Encapsulamento
- Listas (ArrayList)
- Método toString()
