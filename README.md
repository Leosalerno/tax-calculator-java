## 📌 Sobre o Projeto

Calculadora de impostos desenvolvida em **Java**, utilizando conceitos de **Programação Orientada a Objetos (POO)**.

O programa permite cadastrar contribuintes e calcular automaticamente os impostos com base em diferentes tipos de renda e possíveis abatimentos.

---

## 🧠 Funcionamento

- O usuário informa **quantos contribuintes deseja cadastrar**
- Para cada contribuinte são informados:
  - **Renda anual com salário**
  - **Renda anual com prestação de serviços**
  - **Renda anual com ganho de capital**
  - **Gastos médicos**
  - **Gastos educacionais**
- O sistema calcula:
  - **Imposto bruto**
  - **Abatimento**
  - **Imposto final devido**
